# homebrew-tap

Homebrew tap for macOS apps by [choru-k](https://github.com/choru-k).

## Install

```bash
brew tap choru-k/tap
```

## Casks

| Cask | Description | Version |
|------|-------------|---------|
| [bex](https://github.com/choru-k/bex) | Grammar and expression checker powered by AI | 0.1.0 |
| [claude-zellij-whip](https://github.com/choru-k/claude-zellij-whip) | Smart macOS notifications for Claude Code in Zellij | 0.1.0 |
| [mungmung](https://github.com/choru-k/mungmung) | Native macOS stateful notification manager with CLI | 0.1.0 |
| [pastefence](https://github.com/choru-k/PasteFence) | Masks sensitive information in clipboard text using local LLM | 1.0.4 |

```bash
brew install --cask choru-k/tap/bex
brew install --cask choru-k/tap/claude-zellij-whip
brew install --cask choru-k/tap/mungmung
brew install --cask choru-k/tap/pastefence
```
